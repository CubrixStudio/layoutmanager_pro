## Presentation rapide

Layer Manager Pro transforme l'editeur de textures de Blockbench en un veritable editeur d'images multi-calques inspire de Photoshop. Il ajoute :

- **Calques complets** : creation, duplication, import d'images, verrouillage, opacite, modes de fusion
- **Groupes/dossiers** : organisez vos calques en groupes avec visibilite, opacite et verrouillage collectifs
- **Masques non-destructifs** : masques de calque et de groupe (blanc = visible, noir = masque)
- **Filtres empilables** : 8 filtres avec intensite reglable, ordre modifiable, activation/desactivation individuelle
- **Fusion avancee** : merge visible, merge selected, merge down, flatten all
- **Edition externe** : editez un calque dans votre editeur d'images prefere avec synchronisation automatique
- **Integration Photoshop** : export/import PSD complet avec tous les calques
- **Drag & Drop** : reordonnez calques, groupes et filtres par glisser-deposer
- **Persistance** : sauvegarde automatique dans le projet et en localStorage

## Prerequis

- Blockbench 4.9.0 ou superieur
- Mode Paint actif
- Desktop ou Web (l'edition externe necessite la version Desktop)

## Demarrage rapide

1. Installez le plugin (voir [Installation](Installation))
2. Passez en mode **Paint**
3. Selectionnez ou creez une texture
4. Le panneau **Layer Manager Pro** apparait dans la barre laterale droite
5. Commencez a creer des calques avec `Ctrl+Shift+N`
